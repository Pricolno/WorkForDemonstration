## В помощь 



🔴 [Хендбук от ШАД и Яндекса](https://education.yandex.ru/handbook/ml)
Учебник поможет изучить классическую теорию и тонкости реализации алгоритмов, пройдя путь от основ машинного обучения до тем, которые поднимаются в свежих научных статьях.

🔴 [Онлайн-лекции лектория ФПМИ МФТИ](https://www.youtube.com/@lectory_fpmi/search)
Подойдут любителям лекционного формата для более внимательного ознакомления с моделями машинного обучения.

🔴 [Канал Deep Learning School](https://www.youtube.com/channel/UCFTNoZYjkg-3LZTHrHfV1nQ) и их [курсы на Stepik](https://stepik.org/course/124069/promo?auth=registration)
Для нарабатывания практики с фокусом на DL, от основ до продвинутых методов.

🔴 [Книга по дизайну ML-систем (приложений) с практическими примерами](https://www.amazon.com/Designing-Machine-Learning-Systems-Production-Ready/dp/1098107969)
Будет полезна тем, кто хочет разобраться с имплементацией ML-решений в прод.

🔴 [Курсы по ML от Стэнфорда](https://www.coursera.org/specializations/machine-learning-introduction?isNewUser=true)
Для тех, кто хочет узнать первоисточники многих других курсов в русскоязычном пространстве.
