Написать декораток, которые кеширует значение результат функции

```python

cache = dict()
def cache(func):
    def wrapper(*args, **kwargs):
        hash_ = hash(*args, **kwargs)
        if hash_ in cache:
            return cache[hash_]
        cache[hash_] = func(*args, **kwargs)
        return cache[hash_]
        
@cache
def sum(a, b):
    return a + b
```

Итераторы vs генераторы 

итератор имеет методы iter(), next()
генератор частный случай итераторы, не хранит все данные

```
def gen(i):
    for k in range(i):
        yield k

it = iter(gen)
g = next(it) # 0


def fib(n):
    a = 0
    yield a
    b = 1
    yield b
    for i in range(2: n + 1):
        c = a + b
        yield c
        b = a
        a = c
```
