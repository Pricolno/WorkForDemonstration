Задача: 
```python3
def foo(x, a = []): # danger!!!
    if a > 10:
        x += 10
    else:
        x *= 10
    a.append(x)
    return a

print(foo(2))
> [20]
print(foo(20))
> [20, 30]
```

Необязательные переменные, тобишь с значением по умолчанию - их дефолтные значения создаются один раз, во время определниии функции. 

> 

```python 
t = ('kek', 0, [1, 2, 3])
t[1] = 4 # изменяем неизменяемый обьект, у него ссылка изменится
t[2].append(23) # изменим сам обьект, но ссылка останется такая же 
```
